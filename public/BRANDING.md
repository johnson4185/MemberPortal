# MemberConnect Branding Assets

## Logo & Favicons

This folder contains all branding assets for **MemberConnect**.

### Files

#### Favicons
- **`favicon.svg`** - Modern SVG favicon (recommended, scales perfectly)
- **`favicon.ico`** - Traditional ICO format for legacy browser support
- **`apple-touch-icon.png`** - Apple device home screen icon (180x180)

#### Logo
- **`logo.svg`** - Full application logo with shield and connection elements

### Design Elements

**Color Palette:**
- Primary: `#3b82f6` (Blue 500)
- Primary Dark: `#2563eb` (Blue 600)
- Gradient: Linear gradient from Blue 500 to Blue 600

**Logo Concept:**
- **Shield** - Represents security and protection (health insurance)
- **Checkmark** - Symbolizes verified coverage and approved claims
- **Connection Dots** - Represents connectivity between members and healthcare
- **Rounded Square** - Modern, friendly, approachable design

### Usage

#### In HTML/React
```html
<!-- Favicon -->
<link rel="icon" href="/favicon.ico" />
<link rel="icon" type="image/svg+xml" href="/favicon.svg" />

<!-- Apple Touch Icon -->
<link rel="apple-touch-icon" href="/apple-touch-icon.png" />

<!-- Logo -->
<img src="/logo.svg" alt="MemberConnect" />
```

#### Theme Color
```html
<meta name="theme-color" content="#3b82f6" />
```
